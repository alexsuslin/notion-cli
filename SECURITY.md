# Security Policy

## Reporting

If you find a security issue, please report it privately to the maintainer instead of opening a public issue.

Until a dedicated security contact is published, use a private GitHub communication channel associated with the repository owner.

## Scope

Please report issues involving:

- token leakage
- unsafe handling of Notion credentials
- unexpected mutation of Notion content
- command execution vulnerabilities

## Expectations

- Reports will be reviewed privately.
- Reasonable effort will be made to reproduce and fix valid issues.
- Please avoid public disclosure until a fix or mitigation is available.
